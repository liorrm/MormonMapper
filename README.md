MormonMapper
============

the most thoughtful and interactive map of Mormon/LDS temples on the web


###Languages###

####Ruby & JavaScript####


###Frameworks and Libraries###

####Ruby####
- Sinatra (Rack web server)
- Nokogiri (HTML & XML parsing)

####JavaScript####
- jQuery (goes without saying, I know...)
- Leaflet.js
- Moment.js
- Lightbox.js


###APIs###
- Flickr
- Mapbox
